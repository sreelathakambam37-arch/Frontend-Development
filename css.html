CSS

CSS stands for Cascading Style Sheets.It is a stylesheet language used to describe the presentation of a document written in HTML or XML.CSS defines how elements should be displayed on screen, on paper, or in other media.

CSS is made up of rules that consist of a selector and a declaration block. The selector specifies which HTML elements the rule applies to, while the declaration block contains one or more declarations separated by semicolons. Each declaration includes a property and a value.
